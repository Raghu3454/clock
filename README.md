# Digital Alarm Clock

## Description
This is a simple Digital Alarm Clock project built using Python.
It allows users to set an alarm and get notified at the specified time.

## Technologies Used
- Python
- Tkinter
- DateTime
- Threading

## Features
- Set alarm time
- Real-time clock display
- Alarm notification

## How to Run
1. Make sure Python is installed
2. Run the Python file using:
   python clock.py
3. Set the alarm time and wait for notification

## Learning Outcome
This project helped me understand Python GUI, time handling, and threading.
